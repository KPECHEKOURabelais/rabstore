EXERCICE 1 :

  1. Ecrire un programme qui demande à l’utilisateur de saisir un entier N et qui affiche le nombre de nombres premiers inférieurs ou égaux à N.
  2. On considère la suite hongroise : u(0)=a (a entier). Si u(n) pair alors u(n+1)=u(n)/2 sinon u(n+1) = 3*u(n)+1. Pour toutes les valeurs a, il existe un entier N tel que u(N)= 1 (conjecture admise). Ecrire un programme qui demande à l’utilisateur de taper a et qui affiche toutes les valeurs de u(n) de n=1 a n=N.
  3. Ecrire un programme qui saisit 2 tableaux de 10 entiers a et b qui doivent être triés dans l’ordre croissant. Le programme devra tout d’abord vérifier que les deux tableaux sont triés. Le tableau c est un tableau de 20 entiers. Le programme doit mettre dans le tableau c la fusion des tableaux a et b. Le tableau c devra contenir les éléments de a et ceux de b et devra être trié. Le programme affiche ensuite le tableau c.

EXERCICE 2 :

  1. Ecrire une fonction estPremier ayant en paramètre un entier et qui renvoie un booléen : true si l’entier est premier et false sinon.
  2. Ecrire une fonction nPremier ayant comme paramètre un entier n et qui renvoie le nième nombre premier ; cette fonction utilisera la fonction du estPremier
  3. Tester la fonction nPremier dans la fonction main.
  4. Donner le résultat lorsqu’on appelle cette fonction avec le paramètre 25
    NB : Prototyper les deux fonctions.

Correction : 

#include < iostream > 
using namespace std;
bool estPremier(int n);
int nPremier(int n);
int main(){
    int n(0),choix(0);
    cout<<"Entrer un entier : \n";
    cin>>n ;
    cout<<"Le nombre premier occupant la position "<< n<<" est "<< nPremier(n)<<".\n";
}
bool estPremier(int n){
    int i;
    for(i=2;i<=(n-1);i++){
       if(n%i!=0){
           return true;
       }
       else{
           return false;
       }
   }
}
int nPremier(int n){
    int p(0),i(0);
    do{
       p++;
       if(estPremier(p)){
         i++;
       }
    }while(i!=n);
    return p;
}
